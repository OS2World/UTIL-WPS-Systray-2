 Systray/2                  
 History changes.         
____________________________________________________________
                                   Team OS/2 Russian project

 

1.0 alpha 4 (15.08.2000)
------------------------
 Добавлено изменение размера для разных графических
 режимов, режимы Float on top и запрет перемещения по экрану;
 В модуль SystrayTaskbarClass внесена поддержка фильтров
 (можно исключать ненужные задачи из списка работающих задач).
 Написан простенький скрипт установки с базовым вариантом конфигурации.

 В модуле SystrayClock можно изменять цвет фона и шрифт путем
 drag-and-drop'а из палитр OS/2.
 


1.0 alpha 3 (21.07.2000)
------------------------
 То же, что и alpha 2, добавлено описание Tray API,
 изменено строение архива (все части, включая исходные 
 тексты, внесены в один архив)и поправлено оформление
 описаний и документации. 


1.0 alpha 2 (21.07.2000)
------------------------
 Улучшения в модуле Taskbar (список задач); 
 Добавление всплывающих подписей;
 Tray API научился изменять иконку программы для отображения в Tray) 


1.0 alpha (20.07.2000)
----------------------
 первя выпущенная в свет версия.
